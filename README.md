# PHP Native CRUD System

## 📌 Description
A PHP Native backend project built from scratch without any frameworks.  
The system includes full CRUD operations, authentication, pagination, and form validation.

---

## 🚀 Features

- User Authentication System (Login / Logout)
- Session & Cookie-based login system
- Full CRUD Operations (Create, Read, Update, Delete)
- Pagination system for handling large data
- Server-side validation for secure data handling
- Category management (if included)
- Clean separation between logic and UI
- Multi Delete (Delete multiple records at once)
- Multi Upload (Upload multiple files/images at once)
---

## 🛠️ Technologies Used

- PHP Native
- MySQL Database
- HTML / CSS
- Bootstrap

---

## 🔐 Security Features

- Server-side validation
- Protected login system
- Secure session handling
